# AI Receipt Scanner

## Overview
This project is an AI-powered receipt scanner built using Python and Flask. Users can upload receipt images, and the system extracts important fields using OCR.

## Features
- Upload receipt images
- Extract store name, date, total amount, and currency
- Display uploaded receipt preview
- Show extracted OCR text

## Technologies Used
- Python
- Flask
- Tesseract OCR
- pytesseract
- HTML/CSS
